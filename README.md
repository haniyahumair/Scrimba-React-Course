# Scrimba React Course – Projects & Coursework

This repository contains my projects and practical work from the **Scrimba React Course** (which is completely free).

Before discovering Scrimba, I used to watch 4–5 hour long YouTube tutorials.  
By the end of them, I’d forgotten half of what I learned.

It felt productive in the moment… but the information just didn’t stick.

---

## 🚀 What Makes Scrimba Different?

Scrimba isn’t just a video platform.

You can pause **any lesson** and edit the code directly inside the tutorial.

- No switching tabs  
- No copying and pasting  
- No passive watching  

Just hands-on learning.

Because it’s interactive, I actually retain what I’m learning instead of just consuming content. It forces you to think, experiment, and apply concepts in real time — which makes a huge difference.

---

## 💡 What I’ve Learned So Far

Through building these projects, I’ve strengthened my understanding of:

- Reusable components

- Avoiding hardcoded data

They may be small projects, but they build real clarity.
